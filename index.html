Importação da CAIXA

Use a extensão para capturar as OS abertas, baixar os protocolos e importar o pacote de dados.

Fluxo recomendado

1. Abra o portal da CAIXA já logado e vá até a lista de OS abertas.
2. Na extensão, capture a lista e baixe os protocolos em lote.
3. Neste sistema, importe o arquivo JSON exportado pela extensão.
4. Depois importe os PDFs baixados para completar o cadastro com todos os campos do protocolo.
Esta primeira integração já evita retrabalho: captura referências, tipo, data, situação, links de protocolo e prepara a base para importação em lote.
